# 소개
메일 라이브러리를 이용하여 메일을 보내는 방법을 공부한 것입니다.<br>
-> JavaMail, CommonMail
<br>
# 예제
**네이버** : SMTP를 이용하여 보내며 보안접속 방식을 STARTTLS (port : 587) 방식을 사용하였습니다.<br><br>
 **다음**  : SMTP를 이용하여 보내며 보안접속 방식을 SSL (port : 465) 방식을 사용하였습니다.<br>
